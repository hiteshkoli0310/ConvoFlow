# ConvoFlow

ConvoFlow is a modern, real-time chat application designed to bridge language barriers. With built-in translation features, users can communicate seamlessly regardless of their native language.

## ğŸ“¦ Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/hiteshkoli0310/ConvoFlow.git
    ```
2.  Install dependencies for server and client:
    ```bash
    cd server
    npm install
    cd ../client
    npm install
    ```
3.  Set up environment variables (create `.env` files in server and client folders).
4.  Run the development servers:

    ```bash
    # In server directory
    npm run dev

    # In client directory
    npm run dev
    ```
