Welcome to **ConvoFlow**, a modern real-time messaging platform designed to make global communication seamless. With built-in real-time translation, you can chat with anyone, anywhere, regardless of the language they speak.

## What is ConvoFlow?

ConvoFlow is an open-source chat application built to demonstrate the power of real-time communication combined with AI-driven translation. It features a polished, glassmorphism-inspired UI that provides a premium user experience.

Whether you are looking to connect with friends across the globe or just want to explore a modern MERN stack application, ConvoFlow is the perfect playground.

## Getting Started

Follow these steps to set up ConvoFlow locally on your machine.

### Prerequisites

- Node.js installed
- MongoDB installed or a MongoDB Atlas URI

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/hiteshkoli0310/ConvoFlow.git
   ```

2. **Install Dependencies**

   ```bash
   # Install server dependencies
   cd server
   npm install

   # Install client dependencies
   cd ../client
   npm install
   ```

3. **Environment Setup**
   Create `.env` files in both `server` and `client` directories with your configuration (DB URI, JWT Secret, etc.).

4. **Run the App**

   ```bash
   # Run server
   cd server
   npm run dev

   # Run client
   cd client
   npm run dev
   ```
